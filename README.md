# DiscordToDiary

「日記は、ちゃんとした文章で意味のあることを書かなくてはいけない気がする」

そんな気持ちから日記を書くハードルが上がり、続かなくなってしまうことがあると思います。

DiscordToDiaryは、「もっと気軽に毎日日記をつけたい」という方に向けた、**Discord連携型の一言日記サービス**です。

このサービスでは、ブログの管理画面にログインする必要はありません。
Discordの「#日記投稿チャンネル」に一言投稿するだけで、その内容が自動的にあなたの日記として保存されます。

「チャットに投稿した一言」を日記として集めることで、日記を書くハードルを下げ、気軽に日記を続けることができます。

また、「#日記投稿チャンネル」に書き込むことでFBCメンバーが日々どのように過ごしているかを共有することができます。

たくさんの一言を「#日記投稿チャンネル」に投稿しましょう！
